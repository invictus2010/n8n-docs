# Figma

You can use these credentials to authenticate the following nodes:

  • Figma Trigger (Beta)

# Prerequisites

Create a Figma (opens new window) account.

# Using API Key

  1. Log in to Figma (opens new window).
  2. Click on the avatar on the top-right and select Settings.
  3. Scroll down to Personal access tokens, enter a name for your token in the input field and confirm by pressing Enter.
  4. Click on Copy this token.
  5. Paste the API key in the Access Token field of the Figma API credentials screen in n8n.
  6. Click the Save button to save your credentials in n8n.
